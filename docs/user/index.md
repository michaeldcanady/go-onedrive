# Introduction to odc

**odc** (OneDrive CLI) is a Unix-style command-line tool designed to interact with Microsoft OneDrive. It provides a terminal-native experience for managing your files, similar to standard GNU/Linux coreutils like `ls`, `cp`, `mv`, and `rm`.

## Key Features

- **Unix-Style Commands:** Use familiar commands like `ls`, `mkdir`, `cp`, `mv`, and `rm` on your OneDrive files.
- **Profile Management:** Manage multiple OneDrive accounts (e.g., personal, work, school) using profiles.
- **Drive Aliasing:** Easily switch between different drives (e.g., your personal drive, shared libraries, or specific SharePoint sites).
- **Interactive Authentication:** Simple login via your web browser or device code.
- **Filtering and Sorting:** Powerful options for listing files, including recursive views, tree views, and various sorting criteria.
- **File Transfers:** Upload and download files and directories with ease.
- **Direct Editing:** Open and edit OneDrive files directly in your local editor (e.g., `vi`, `nano`, `vscode`).

## Why use odc?

If you spend your time in the terminal, switching to a web browser to manage your OneDrive files can be disruptive. `odc` bridges this gap, allowing you to stay in your workflow while managing your cloud storage as if it were a local filesystem.
